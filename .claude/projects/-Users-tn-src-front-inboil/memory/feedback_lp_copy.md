---
name: LP copy should be compelling, not terse
description: Landing page copywriting should be warm and engaging, not bullet-point style influenced by conversation brevity
type: feedback
---

LP and user-facing marketing copy should be written with care and warmth — not in the terse, minimal style used for developer conversation.

**Why:** The concise communication style between user and Claude is for efficiency. Applying that same style to landing page copy makes it feel lifeless and unappealing. LP text needs to sell the product and create excitement.

**How to apply:** When writing LP copy, feature descriptions, taglines, or any user-facing marketing text, write in a warm, inviting, slightly playful tone. Take time to craft sentences that create curiosity and delight. Don't just list facts — tell a story.
