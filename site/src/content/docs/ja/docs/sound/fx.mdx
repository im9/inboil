## FX Pad

FX Pad は、ドラッグ可能な XY パッドとして配置された **4 つのセンドエフェクト**を提供します。各エフェクトには 3 つのバリエーションがあり、エフェクト名をタップして切り替えられます。

<PlaygroundFxPad client:visible />

### エフェクト一覧

| エフェクト | バリエーション |
|-----------|---------------|
| **Reverb** | Room / Hall / Shimmer |
| **Delay** | DIGI（ピンポン）/ DOT.8（付点 8 分）/ TAPE（ウォーム LP フィルター） |
| **Glitch** | CRUSH（ビットクラッシュ）/ REDUX（ダウンサンプル）/ STTR（スタッター） |
| **Granular** | CLOUD（グレインシャワー）/ FREEZE（オートフリーズ）/ STRCH（タイムストレッチ） |

### トラックごとのセンド

各トラックには、各エフェクトへの独立したセンドレベルがあります（Dock パネルの VERB / DLY / GLT / GRN ノブ）。上にドラッグしてセンド量を増やします。

### XY コントロール

- **X 軸** — タイム／サイズ（リバーブサイズ、ディレイタイム、グレインサイズ）
- **Y 軸** — キャラクター／フィードバック（リバーブダンピング、ディレイフィードバック、グレイン密度）
- パッドを**タップ**でエフェクトのオン／オフを切り替え
- パッドを**長押し**（400ms）で**バブルメニュー**が表示され、フレーバーを切り替え
- Granular パッドはエフェクトが ON のとき、長押しで**フリーズ**を切り替え

## EQ

EQ は **3 バンドイコライザー**とフィルタースイープノードを搭載しています。

<PlaygroundFilterView client:visible />

### バンド

| バンド | 説明 |
|--------|------|
| **LOW** | 低域 — ピーキングまたはシェルフ（ダブルタップで切替） |
| **MID** | 中域 — ピーキング EQ |
| **HIGH** | 高域 — ピーキングまたはシェルフ（ダブルタップで切替） |
| **Filter** | LP/HP スイープ |

ノードをドラッグしてゲインと周波数を調整します。

## Master

Master セクションは最終出力段を制御します。

<PlaygroundMasterView client:visible />

### XY パッド

| パッド | X 軸 | Y 軸 |
|--------|------|------|
| **COMP** | Threshold | Ratio |
| **DUCK** | Depth | Release |
| **RET** | Reverb リターンレベル | Delay リターンレベル |

### VU メーター

VU メーターはリアルタイムの **L/R ピークレベル**とゲインリダクション（GR）を表示します。クリッピングを防ぐため、ピークを 0 dB 以下に保ちましょう。

## インサート FX

各トラックには **2 スロットの直列インサートチェーン**があり、センドエフェクトの前にインラインで処理されます（Voice → Slot 1 → Slot 2 → Send/Mix）。

| 設定 | 選択肢 |
|------|--------|
| **タイプ** | OFF / Reverb / Delay / Glitch |
| **フレーバー** | センド FX と同じ（Room/Hall、Digital/Dotted/Tape、Bitcrush/Redux） |
| **MIX** | ドライ／ウェットブレンド（0〜100%） |
| **X / Y** | タイプ依存（例: リバーブは Size/Damp、ディレイは Time/FB） |

各スロットは独立しており、Reverb → Delay、Delay → Glitch などの組み合わせが可能です。ベロシティレーンの **INS** タブから、すべてのインサートパラメータ（MIX、X、Y × 各スロット）のステップごとの **P-Lock** に対応しています。

両スロットを OFF にすると UI が1行に折りたたまれます。

## 信号フロー

```
Voice → Insert FX (Slot 1 → Slot 2) → トラックごとの Send ──► Reverb / Delay / Glitch / Granular
                                                                       │
                                                                 FX Return (RET)
                                                                       ↓
Voice ──► Sidechain Ducker ──► Bus Compressor ──► 3-Band EQ
              ──► DJ Filter ──► Break Gate ──► Master Gain
              ──► Peak Limiter ──► Output
```
