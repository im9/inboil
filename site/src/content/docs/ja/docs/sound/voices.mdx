## 概要

各トラックには**ボイス**があります。ボイスとは、ステップがトリガーされたときにオーディオを生成するサウンドエンジンです。INBOIL には**ドラム**、**シンセ**、**サンプラー**の 3 種類のボイスがあります。ドラムとシンセはリアルタイム合成、Sampler はマルチサンプルゾーンマッピングと 8 ボイスポリフォニーによるサンプル再生です。

トラックを選択し、**Dock パネル**でボイスのパラメータを編集できます：

## ドラムボイス

すべてのドラムボイスは共通の合成エンジンを使い、**OSC**（トーンオシレータ）、**NOISE**、**METAL**（メタリックオシレータ）、**AMP** の 4 つのパラメータグループで構成されます。

| ラベル | ボイス | キャラクター |
|--------|-------|-------------|
| **909K** | 909 Kick | パンチのあるアナログキック |
| **808K** | 808 Kick | ディープなサブベースキック |
| **SNARE** | Snare | ノイズ＋トーンのスネア |
| **CLAP** | Clap | レイヤードハンドクラップ（バーストモード） |
| **C.HH** | Closed HH | 短いメタリックハット — オープンをチョーク |
| **O.HH** | Open HH | リンギングするメタリックハット |
| **CYM** | Cymbal | 606 スタイルのノイズ＋サイン波＋BP フィルター |
| **TOM** | Tom | ピッチドドラム |
| **RIM** | Rimshot | シャープなトランジェント |
| **BELL** | Cowbell | メタリックベル |
| **SHKR** | Shaker | ノイズベースのシェイカー |
| **FMD** | FM Drum | FM パーカッション — 6 マシン (kick/snare/metal/perc/tone/chord) |
| **CRSH** | Crash | クラッシュシンバル |
| **RIDE** | Ride | ライドシンバル |

### 主なパラメータ

| パラメータ | グループ | 説明 |
|-----------|---------|------|
| **PSTRT / PEND** | OSC | ピッチの開始→終了（スイープ範囲） |
| **PDCY** | OSC | ピッチスイープ速度 |
| **TONE** | OSC | トーンオシレータレベル |
| **NOIS** | NOISE | ノイズレベル |
| **FREQ / Q** | NOISE | ノイズフィルター周波数＆レゾナンス |
| **METL / MFRQ** | METAL | メタリックオシレータのレベル＆周波数 |
| **DCY** | AMP | 音量ディケイ |
| **DRIV** | AMP | ディストーション量 |
| **CLCK** | AMP | トランジェントクリック |
| **BRST / GAP** | AMP | バースト回数＆間隔（クラップのレイヤリング） |

ドラムボイスごとに使用するパラメータの組み合わせが異なります。例えば KICK は主に OSC + AMP、C.HH は METAL + NOISE を活用します。

### FMD（FM ドラム）

6 種類のマシンアルゴリズムを切り替えられる FM ベースのドラムシンセサイザー。8 つのマクロパラメータが内部で FM オペレーターの比率、変調深度、エンベロープに変換されます。

- **MCH** — マシンタイプ：KICK / SNARE / METAL / PERC / TONE / CHORD
- **CLR** — カラー（倍音の明るさ）
- **SHP** — シェイプ（波形キャラクター）
- **SWP** — スイープ（ピッチエンベロープ深度）
- **CNTR** — コンター（エンベロープカーブ）
- **PNCH** — パンチ（トランジェントアタック）
- **DCY** — ディケイ
- **TONE** — トーンミックス

6 種類のマシンタイプにわたる 21 のファクトリープリセット。TONE と CHORD マシンはピッチ対応で、ピアノロール入力が可能です。

## シンセボイス

### BASS（303 スタイルアシッドベース）
Roland TB-303 にインスパイアされたモノフォニックベースシンセ。
- **CUT** — フィルターカットオフ
- **MOD** — フィルターエンベロープ深度
- **RESO** — フィルターレゾナンス
- **DCY** — 音量ディケイ
- **DRIV** — ディストーション

BASS ボイスは**スライド**（SLD）に対応しており、ポルタメントグライドが可能です — アシッドパターンに欠かせない機能です。

### BASS — Analog（アナログベース）
シンプルなパラメータ構成のモノフォニックベースシンセ。
- **CUT / MOD / RESO** — フィルターセクション
- **DCY** — 音量ディケイ

### LEAD（アナログシンセ）
モノフォニックのリードシンセ。フル ADSR エンベロープとアルペジエーター内蔵。
- **CUT / MOD / RESO / FDCY** — フィルターセクション
- **ATCK / ADCY / SUST / RLS** — アンプ ADSR エンベロープ
- **ARP / RATE / CHRD / AOCT** — アルペジエーター（OFF / UP / DOWN / U-D / RND）

<PlaygroundEnvGraph client:visible />

### FM（4 オペレータ FM シンセ）
4 オペレーターの FM シンセ、8 種類のアルゴリズム搭載。

<PlaygroundAlgoGraph client:visible />

- **ALG** — ルーティングアルゴリズム（8 パターン）
- **R1〜R4** — オペレーター周波数比
- **L1〜L4** — オペレーター出力レベル
- **D1〜D4** — オペレーターディケイ
- **FB** — OP1 セルフフィードバック
- **ATK / REL** — キャリアのアタック＆リリース
- **LFO** — レート、深さ、波形、送り先

ポリモード：**MONO** / **POLY 12** / **WIDE 6** / **UNISON**

### WAVE（ウェーブテーブル）
デュアルオシレータのモーフィングウェーブテーブルシンセ。ユニゾンと LFO 搭載。

<PlaygroundWaveGraph client:visible />

- **WV-A / WV-B** — 各オシレータのウェーブテーブル位置
- **MIX** — オシレータ A/B ミックス
- **COMB** — 合成モード（MIX / FM / RING）
- **CUT / FMOD / RESO / FTYP** — フィルター（LP / HP / BP / NTCH）
- **DRIV** — ポストフィルターサチュレーション
- **UNI / SPRD / WIDE** — ユニゾンボイス数、デチューン幅、ステレオ幅
- **ATCK / DCY / SUST / RLS** — アンプ ADSR エンベロープ
- **LFO 1** — テンポ同期対応の LFO
- **LFO 2** — フリーランニング LFO

ポリモード：**MONO** / **POLY 16** / **WIDE 8** / **UNISON**

### SMPL（サンプラー）
マルチサンプルゾーンマッピング対応の 8 ボイスポリフォニックサンプル再生エンジン。
- **STRT / END** — サンプルの開始・終了位置
- **PTCH** — ピッチシフト（±24 半音）
- **DCY** — 振幅ディケイ
- **REV** — 逆再生
- **CHOP / MODE** — 8/16/32 セグメントにスライス（MAP または SEQ モード）
- **BPM / LOOP / STRC** — BPM 同期、ループ、タイムストレッチ（リピッチまたは WSOLA）

Sampler は**メロディックボイス**です — ピアノロール入力、トランスポーズ、アルペジエーターに対応しています。**Grand Piano** マルチサンプルパックでキーボード全域のリアルなピアノを、ボーカルチョップやループでクリエイティブなサンプリングを楽しめます。

詳細は [Sampler](/ja/docs/sound/sampler/) を参照してください。

## ボイスの切り替え

**Dock タブ**（**DRUM** / **SYNTH** / **SMPL**）でトラックのボイスカテゴリを切り替え、具体的なボイスを選択します。変更は即座に反映されます — 既存のステップはトリガーを保持したまま、新しいサウンドで再生されます。
