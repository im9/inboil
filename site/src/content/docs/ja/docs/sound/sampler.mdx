## 概要

**Sampler** ボイス（SMPL）は、オーディオファイルを読み込んでインストゥルメントとして演奏できるボイスです。スライス、ピッチコントロール、BPM 同期を備えたフル機能のサンプラーです。

Dock パネルの **SMPL** タブで、任意のトラックを Sampler に切り替えられます。

## サンプルの読み込み

### ファイルから
**LOAD** ボタンをタップするか、オーディオファイルをトラックに**ドラッグ＆ドロップ**します。対応フォーマット：WAV、MP3、OGG、FLAC。

### ライブラリから
今後のアップデートで内蔵サンプルライブラリを追加予定です。現時点では、お手持ちのファイルをご利用ください。

## パラメータ

| パラメータ | 説明 |
|-----------|------|
| **START** | サンプルの開始位置 |
| **END** | サンプルの終了位置 |
| **PITCH** | ピッチシフト（半音単位） |
| **DECAY** | 振幅のディケイ |
| **REVERSE** | サンプルを逆再生 |

## CHOP モード

**CHOP** をタップすると、サンプルを均等なセグメントに自動スライスします。パターン内の各ステップが異なるスライスをトリガーします — ブレイクビーツやドラムループに最適です。

チョッピングは非破壊処理です。いつでも元のフルサンプルに戻せます。

## BPM 同期

**BPM sync** を有効にすると、現在のテンポに合わせてサンプルを自動的にストレッチ／圧縮します。これにより、元の BPM に関係なくループがテンポに同期します。

## ヒント

- **サンプルとシンセボイスを重ねる**：1 つのトラックにサンプル、もう 1 つにシンセレイヤーを配置して厚みを追加
- **リバースフィル**：コピーしたパターンで REVERSE を有効にしてトランジションエフェクトに
- **マイクロサンプリング**：START と END を非常に近い位置に設定してグラニュラーテクスチャーを作成
- **ワンショット vs ループ**：短いサンプル（キック、スネアなど）はワンショットとして動作し、長いサンプルはパターンの繰り返し時に自動ループします
