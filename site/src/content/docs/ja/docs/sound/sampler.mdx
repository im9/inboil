## 概要

**Sampler** ボイス（SMPL）は、オーディオファイルを読み込んでインストゥルメントとして演奏できるボイスです。スライス、ピッチコントロール、BPM 同期を備えたフル機能のサンプラーです。

Dock パネルの **SMPL** タブで、任意のトラックを Sampler に切り替えられます。

## サンプルの読み込み

**LOAD** ボタンをタップするか、オーディオファイルを波形エリアに**ドラッグ＆ドロップ**します。対応フォーマット：WAV、MP3、OGG、FLAC（最大 10 MB）。

## パラメータ

| パラメータ | ラベル | 説明 |
|-----------|--------|------|
| **STRT** | Start | サンプルの開始位置（0.0〜1.0） |
| **END** | End | サンプルの終了位置（0.0〜1.0） |
| **PTCH** | Pitch | ピッチシフト（半音単位、±24） |
| **DCY** | Decay | 振幅のディケイ |
| **REV** | Reverse | 逆再生（OFF / ON） |

## CHOP モード

サンプルを均等なセグメントにスライスします。ブレイクビーツやドラムループに最適です。

| パラメータ | ラベル | 説明 |
|-----------|--------|------|
| **CHOP** | Slices | スライス数：OFF / 8 / 16 / 32 |
| **MODE** | Mode | **MAP** — ノートピッチでスライスを選択、**SEQ** — トリガーごとに次のスライスへ進む |

チョッピングは非破壊処理です。CHOP を OFF に戻せば元のフルサンプルに戻せます。

## BPM 同期 & ループ

| パラメータ | ラベル | 説明 |
|-----------|--------|------|
| **BPM** | Sample BPM | 元のサンプルテンポ（0 = OFF）。設定するとプロジェクト BPM に合わせて再生速度を自動調整 |
| **LOOP** | Loop | **1SHT**（ワンショット）または **LOOP**（連続ループ） |
| **STRC** | Stretch | **RPTC**（リピッチ — テンポに伴いピッチも変化）または **WSLA**（WSOLA — ピッチを維持） |

## ヒント

- **サンプルとシンセボイスを重ねる**：1 つのトラックにサンプル、もう 1 つにシンセレイヤーを配置して厚みを追加
- **リバースフィル**：コピーしたパターンで REV を有効にしてトランジションエフェクトに
- **マイクロサンプリング**：STRT と END を非常に近い位置に設定してグラニュラーテクスチャーを作成
- **ブレイクビーツ**：CHOP を 16、MODE を SEQ に設定してクラシックなブレイクビーツパターンに
