**INBOIL** のドキュメントへようこそ。INBOIL はブラウザベースのグルーブボックス / DAW です。インストール不要、登録不要 — 開いたらすぐ演奏できます。

## はじめに

初めての方はチュートリアルから:

1. [最初のビートを作る](/ja/docs/getting-started/first-beat/) — 再生、ステップ入力、ノブ操作
2. [音色を変える](/ja/docs/getting-started/adding-sounds/) — ドラムやシンセを調整
3. [シーンを作る](/ja/docs/getting-started/arrangement/) — パターンをグラフで構成
