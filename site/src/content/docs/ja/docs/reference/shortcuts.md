## グローバル

| ショートカット | アクション |
|---------------|-----------|
| **Space** | 再生／停止 |
| **Ctrl+Z** | 元に戻す |
| **Ctrl+Shift+Z** | やり直し |
| **Escape** | シートを閉じる／選択解除 |
| **Delete** | 選択したノードまたはエッジを削除 |
| **Ctrl+C** | パターンをコピー |
| **Ctrl+V** | パターンをペースト |

## ステップグリッド

| ショートカット | アクション |
|---------------|-----------|
| セルを**クリック** | ステップのオン／オフ切り替え |
| セルをまたいで**ドラッグ** | 複数ステップをペイント |
| トラック名を**クリック** | トラックを選択（Dock で開く） |
| ステップカウントを**クリック** | ステップ数を切り替え（2 / 4 / 8 / 16 / 32 / 64） |
| **Shift+ドラッグ**（ベロシティバー） | ベロシティとチャンス編集を切り替え |

## Tracker モード

| ショートカット | アクション |
|---------------|-----------|
| **矢印キー** | セル間を移動 |
| **Enter** | ステップのオン／オフ切り替え（SLD 列：スライド切り替え） |
| **Delete / Backspace** | ステップをクリア |
| **PageUp / PageDown** | 8 行上下にジャンプ |
| **[ / ]** | 前後のトラックに切り替え |
| **0–9** | ベロシティ設定（VEL 列）またはチャンス設定（CHN 列） |
| **C–B** | ノート名を入力（メロディトラック、NOTE 列） |
| **#** | シャープ／フラットを切り替え |
| **Shift+上 / Shift+下** | オクターブ上下 |

## ピアノロール

| ショートカット | アクション |
|---------------|-----------|
| 空のセルを**クリック** | ノートを配置 |
| 既存のノートを**クリック** | ノートを削除 |
| ノートを縦に**ドラッグ** | ピッチを変更 |
| ノートを横に**ドラッグ** | ステップ位置を変更 |
| ノートの右端を**ドラッグ** | デュレーションを変更 |
| **右クリック** | ノートを消去（Draw モード） |
| **D** を押し続ける | 一時的に Draw モードに切り替え |
| **E** を押し続ける | 一時的に Eraser モードに切り替え |
| **C** を押し続ける | 一時的に Chord モードに切り替え |
| **▲ / ▼** または**スクロール** | オクターブ窓をシフト |

## Scene ビュー

| ショートカット | アクション |
|---------------|-----------|
| ノードを**クリック** | ノードを選択 |
| **Shift+クリック**（ノード） | 複数選択に追加／除外 |
| **Shift+クリック**（ラベル） | ラベルを複数選択に追加／除外 |
| ノードを**ドラッグ** | 位置変更（複数選択時は全選択を移動） |
| 背景を**ドラッグ** | 矩形選択（ノードとラベルをボックス選択） |
| エッジを**クリック** | エッジを選択 |
| ノード右側の**ハンドルをドラッグ** | エッジ描画を開始（ターゲットにドラッグ） |

| 空きスペースを**右クリック** | バブルメニュー（ノード／ラベルを追加） |
| **Delete** | 選択した要素を削除 |
| **Escape** | 選択解除 |
| **Ctrl+C** | ノード／サブグラフをコピー |
| **Ctrl+V** | ペースト |
| **矢印キー上下**（エッジ選択中） | エッジの順序を変更（再生順） |
| **スクロール／ピンチ** | ズーム |
| **ミドルドラッグ** または **Ctrl+ドラッグ** | キャンバスをパン |
| **Alt+A / D / W / S**（2ノード以上） | 左 / 右 / 上 / 下に整列 |
| **Alt+X / E**（2ノード以上） | 水平 / 垂直に中央揃え |
| **Alt+Shift+X / E**（2ノード以上） | 水平 / 垂直に均等配置 |

## バーチャルキーボード（VKBD）

パターンツールバーの **VKBD** ボタンで有効化します。

| キー | ノート |
|------|--------|
| **A S D F G H J K L ;** | C D E F G A B C D E（クロマチック） |
| **Z** | オクターブダウン |
| **X** | オクターブアップ |
| **1–0** | ベロシティ（1=低、0=最大） |

## MIDI

Web MIDI API 経由で USB/BLE MIDI キーボードに対応しています。入力デバイスとチャンネルルーティングは **SYSTEM → MIDI** で設定します。
