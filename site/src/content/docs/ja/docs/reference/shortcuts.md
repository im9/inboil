title: キーボードショートカット

---

## グローバル

| ショートカット | アクション |
|---------------|-----------|
| **Space** | 再生／停止 |
| **Ctrl+Z** | 元に戻す |
| **Ctrl+Shift+Z** | やり直し |
| **Escape** | パネルを閉じる／選択解除 |
| **Delete** | 選択したノードまたはエッジを削除 |
| **Ctrl+C** | パターンをコピー |
| **Ctrl+V** | パターンをペースト |

## ステップグリッド

| ショートカット | アクション |
|---------------|-----------|
| セルを**クリック** | ステップのオン／オフ切り替え |
| セルをまたいで**ドラッグ** | 複数ステップをペイント |
| トラックラベルを**クリック** | トラックを選択（Dock で開く） |
| ステップカウントを**クリック** | ステップ数を変更（2〜64） |

## Tracker モード

| ショートカット | アクション |
|---------------|-----------|
| **矢印キー** | セル間を移動 |
| **Enter** | 選択したセルを編集 |
| **Tab** | 次の列に移動 |
| **Delete** | セルの値をクリア |
| **Escape** | 編集をキャンセル |

## ピアノロール

| ショートカット | アクション |
|---------------|-----------|
| 空のセルを**クリック** | ノートを配置 |
| 既存のノートを**クリック** | ノートを削除 |
| ノートを縦に**ドラッグ** | ピッチを変更 |
| ノートの端を**ドラッグ** | デュレーションを変更 |

## Scene ビュー

| ショートカット | アクション |
|---------------|-----------|
| ノードを**クリック** | ノードを選択 |
| ノードを**ドラッグ** | 位置変更 |
| エッジを**クリック** | エッジを選択 |
| ノードの端から**ドラッグ** | 接続を作成 |
| 空きスペースを**右クリック** | コンテキストメニュー（ノード／ラベルを追加） |
| **Delete** | 選択した要素を削除 |
| **スクロール／ピンチ** | ズーム |
| **ミドルドラッグ** | キャンバスをパン |

## バーチャルキーボード（VKBD）

**VKBD** ボタンで有効化します。

| キー | ノート |
|------|--------|
| **A S D F G H J K L ;** | C D E F G A B C D E（クロマチック） |
| **Z** | オクターブダウン |
| **X** | オクターブアップ |
| **1–0** | ベロシティ（1=低、0=最大） |

## MIDI

USB および BLE MIDI キーボードに対応しています。入力デバイスとチャンネルルーティングは **SYSTEM → MIDI** で設定します。
