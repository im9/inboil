スウィープノードを使うと、**リアルタイムでパラメータの変化を録音**できます。シーン再生中にパッドやノブを操作すると、そのジェスチャーがオートメーションカーブになります。

## 使い方

1. **録音をアーム** — シーンビューのスウィープノードの ● ボタン、またはスウィープエディタのツールバーをタップ
2. **任意のパラメータを操作** — FX パッド、マスターパッド、EQ ノード、トラックノブをドラッグ。再生が自動的に開始し、録音が始まります
3. **パフォーマンス** — 通常通りパッドを動かしたりエフェクトを切り替えたりします。すべての動きがキャプチャされます
4. **停止** — ● をもう一度タップするか、再生を停止。カーブがスウィープノードに書き込まれます

スウィープノードは録音時に**自動生成**されます — 事前に作成する必要はありません。

## 録音されるもの

| 操作 | 記録形式 |
|------|----------|
| FX パッドをドラッグ（verb, delay, glitch, granular X/Y） | 連続カーブ |
| マスターパッドをドラッグ（comp, duck, ret, sat X/Y） | 連続カーブ |
| フィルターパッドをドラッグ（cutoff, resonance） | 連続カーブ |
| EQ ノードをドラッグ（freq, gain, Q） | 連続カーブ |
| トラック volume/pan ノブをドラッグ | 連続カーブ |
| ボイスパラメータノブをドラッグ | 連続カーブ |
| センドレベルノブをドラッグ | 連続カーブ |
| FX on/off 切替 | トグルカーブ |
| FX ホールド切替 | トグルカーブ |
| トラックミュート切替 | トグルカーブ |

## スウィープエディタ

スウィープノードを選択 → Dock パネル → 編集でエディタを開きます。ダークゾーンのエディタには以下が表示されます:

- **カーブセクション** — 録音された各パラメータのミニカーブ
- **トグルセクション** — カラーブロック（ON = アクセントカラー、OFF = ダーク）
- **キャンバス** — タイムラインと再生カーソル、スクロール/ピンチでズーム

### 編集

| 操作 | 効果 |
|------|------|
| リストのカーブをクリック | 編集対象として選択 |
| コントロールポイントをドラッグ | カーブ上で位置を変更 |
| ポイントをダブルクリック | 削除 |
| カーブの線上をクリック | 新しいポイントを追加 |
| Shift+キャンバスをドラッグ | ポイントの範囲選択 |
| Delete/Backspace で範囲選択中 | 範囲内のポイントを削除 |
| 範囲のエッジをドラッグ | トリム範囲を調整 |
| ポイント選択 → T/V ノブ | 精密な時間/値の制御 |

### 再生プレビュー

シーン再生中、アンバーのカーソルがキャンバスを横断します。グラデーションの軌跡がカーソルに追従します。

### クリア

**Clear** ボタンで全カーブをフェードアウトして削除（グローバルスウィープデータも含む）。

## グローバル vs チェーンスコープ

パラメータは適切なスコープに自動ルーティングされます:

| スコープ | 対象 | 意味 |
|----------|------|------|
| **グローバル** | Master, FX, EQ, Filter, FX on/off, Hold | シーン全体の再生時間にわたって適用 |
| **チェーン** | トラック volume/pan, ボイスパラメータ, センド, ミュート | 接続されたパターンのチェーン再生時のみ適用 |

## オーバーダブ

再録音は既存データとマージされます:
- **操作したパラメータ**は新しい録音で上書き
- **操作していないパラメータ**は既存のカーブを保持

## 録音プレビュー

録音中、パフォーマンスパッドの上にスパークラインの細いストリップが表示され、キャプチャ中のパラメータのリアルタイムトレースを確認できます。

## シーンビュー

スウィープノードはジェネレーターサイズのカードとして表示されます:
- 録音されたカーブの**プレビュー**
- 接続先の**パターン名**（例: → FROST）
- アクティブなカーブの数

## ヒント

- **リピートノードと併用** — Repeat ×6 ノードがあれば、6リピート分のスウィープを録音できます。リピートノードがなければ、1回の再生分のスウィープになります
- **複数パラメータを重ねる** — 1回の録音パスで複数のパッドを動かして、複雑な変化を作れます
- **パスを分けてオーバーダブ** — まずフィルターを録音し、次にリバーブセンドを別パスで録音。各パスは操作したパラメータのみ上書きします
- **アンドゥ** — Ctrl+Z で録音前の状態に復元
