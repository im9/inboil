スウィープノードを使うと、**時間の経過とともにパラメータを変化させるカーブを描く**ことができます。アシッド的なフィルタースウィープ、ボリュームフェード、複数リピートにわたるテクスチャの変化に最適です。

## 使い方

1. ツールパレットまたはバブルメニューから**スウィープノードを追加**
2. パターンノードに**エッジで接続**
3. スウィープノードを**ダブルタップ**してペイントキャンバスを開く
4. パレットから**パラメータを選択**（例: KICK → Volume、BASS → CUT）
5. キャンバスに**カーブを描く** — X軸は時間（全リピート分）、Y軸はパラメータのオフセット

スウィープはパラメータのベース値に対する**相対オフセット**を適用します。中心線より上は値の増加、下は減少を意味します。

## ペイントキャンバス

キャンバスは接続されたパターンの全リピートにわたるタイムラインを表示します。タイムラインヘッダーのリピート番号（R1, R2, ...）をクリックすると、そのリピートにズームインして細かい制御ができます。

### 描画モード

| モード | 動作 |
|--------|------|
| **Draw** | フリーハンド描画 — ドラッグでカーブを描く（自動スムージング付き） |
| **Point** | クリックでアンカーポイントを配置、ドラッグで移動、ダブルクリックで削除 |

### シェイププリセット

**Shape** ボタンでプリセットのカーブ形状を即座に適用:

| 形状 | 効果 |
|------|------|
| ╱ Ramp Up | 0 から +1 へ徐々に増加 |
| ╲ Ramp Down | 0 から −1 へ徐々に減少 |
| ╱╲ Triangle | ピークまで上昇して戻る（スウェル） |
| ╲╱ V (Dip) | 下がって戻る |
| ⌒ S-Curve | ゆっくり始まり加速する上昇 |

### クリア

**Clear** ボタンで全カーブを削除。

## パラメータ

パレットには各トラックの全パラメータがカテゴリ別に表示されます:

| カテゴリ | 例 |
|----------|-----|
| **ミックス**（○ アウトラインドット） | Volume, Pan |
| **シンセ**（● 塗りつぶしドット） | 全ボイス固有パラメータ（TONE, CUT, RESO, DECAY 等） |
| **センド**（○ アウトラインドット） | Verb, Delay, Glitch, Grain センドレベル |

シンセパラメータはボイスタイプによって異なります — ドラムトラックとベースシンセでは表示されるパラメータが違います。

## 再生

スウィープオートメーションは**シーン再生時のみ**適用されます（パターンループモードでは適用されません）。スウィープは毎ステップでカーブを評価し、リアルタイムでパラメータを更新します。

- 再生中、キャンバスに**プレイヘッド**ラインが移動します
- シーンビューの**スウィープノード**にもカーブプレビュー上をプレイヘッドが移動します
- 接続エッジが再生中にスウィープのアクセントカラーで光ります

再生停止後、全パラメータはオートメーションスナップショットにより**元の値に復元**されます。

## シーンビュー

スウィープノードは generative ノードと同じサイズで表示されます:
- 描いたカーブの**プレビュー**
- 接続先の**パターン名**（例: → FROST）
- アクティブなカーブの数

## ヒント

- **リピートノードと併用** — Repeat ×6 ノードがあれば、6リピート分のスウィープを描けます。リピートノードがなければ、1回の再生分のスウィープになります
- **Volume でミュート** — Volume を −1 まで下げると、スウィープの一部でトラックを実質ミュートできます
- **複数カーブを重ねる** — 同じスウィープノードで異なるパラメータを描くと、複雑な変化を作れます
- **ズームで精密操作** — リピート番号をクリックしてズームイン、Point モードで正確な制御
