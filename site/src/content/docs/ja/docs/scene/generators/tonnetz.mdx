[Tonnetz](https://en.wikipedia.org/wiki/Tonnetz)（和声関係の数学的格子）をナビゲートして**コード進行を生成**します。ネオ・リーマン音楽理論に基づいています。

初期トライアドから始めて、一連の変換を適用して関連するコードを辿ります。各変換は 3 音のうち 2 音を共有する近傍のコードへ移動します。

## 変換

| 操作 | 名前 | 効果 |
|------|------|------|
| **P** | Parallel | 3 度を反転 — メジャー ↔ マイナー（C major → C minor） |
| **L** | Leading-tone | ルートまたは 5 度を半音移動（C major → E minor） |
| **R** | Relative | 関係調に移動（C major → A minor） |

複合操作は複数の変換を同時に適用します：**PL**、**PR**、**LR**、**PLR**。

## パラメータ

| コントロール | 説明 |
|-------------|------|
| **Chord** | 初期トライアド — ルートノート + 性質（major / minor） |
| **Sequence** | 変換操作の順序リスト。**+** で追加、**−** で削除。シーケンスは使い切ると最初からループします。 |
| **STEPS** ノブ | コードあたりのステップ数（1–16） — 次の変換までコードが保持される長さ |
| **Voicing** | コードノートの間隔：**Close**（ルート、3rd、5th）、**Spread**（ルート、3rd+オクターブ、5th）、**Drop2**（ルート、5th、3rd+オクターブ） |

## 例

初期コード：C major。シーケンス：P → L → R。

| ステップ | 変換 | コード |
|---------|------|--------|
| 1–4 | — | C major（C E G） |
| 5–8 | P | C minor（C E♭ G） |
| 9–12 | L | A♭ major（A♭ C E♭） |
| 13–16 | R | F minor（F A♭ C） |
| 17–20 | P（ループ） | F major（F A C） |
