## 1. 再生と停止

**Space** キー（または **▶** ボタン）を押すと再生が始まります。もう一度押すと停止します。カーソルがグリッド上を進み、有効なステップが順に鳴ります。

## 2. ノブを回す

inboil ではほぼすべてのパラメーターをノブで操作します。**上にドラッグ**で値を上げ、**下にドラッグ**で値を下げます。試してみましょう:

<KnobDemo client:visible />

**Shift** を押しながらドラッグすると細かく調整できます。ダブルクリックでデフォルト値にリセットされます。

## 3. ステップを打ち込む

グリッドのセルをタップしてトリガーの ON/OFF を切り替えます。**ドラッグ**で一度に複数のステップを描けます。各行がトラック（KICK、SNARE、HI-HAT など）に対応しています。

下のグリッドには4つ打ちの KICK と HI-HAT パターンがプリセットされています。**▶ Play** を押して音を聴いてから、SNARE を追加したりステップを切り替えてみてください:

<PlaygroundAudioDemo client:load />

### ステップ数

各トラックは独立して **2〜64 ステップ** を持てます。トラック右端のステップ数をタップして変更できます。奇数ステップにするとポリリズムが生まれます — 16 ステップの KICK に対して 5 や 7 を試してみましょう。

## 4. 音色を変える

トラックの**ラベル**（KICK、SNARE など）をタップして選択すると、左側の **Dock パネル**にそのトラックのボイスパラメーターが表示されます:

- **VOL** — トラック音量
- **PAN** — 左右のステレオ定位
- **TUNE** — ピッチの上下
- **DECAY** — 音の余韻の長さ

ボイスの種類（KICK、SNARE、BASS、LEAD、FM、WAVE）ごとに固有のパラメーターがあります。詳しくは[音色を変える](/ja/docs/getting-started/adding-sounds/)をご覧ください。

## 5. ランダマイズ

**RAND** を押すとランダムなパターンが生成されます。気に入らなければ **Ctrl+Z** で元に戻せます。アンドゥは無制限 — すべての編集が記録されています。

## 6. パフォーマンスコントロール

再生中にパフォーマンスボタンを押し続けると、リアルタイムエフェクトがかかります:

<PlaygroundPerfButtons client:visible />

- **FILL** — 押している間、自動ドラムフィル
- **REV** — 逆再生
- **BRK** — リズミックブレイク / スタッター

ボタンを離すと即座に通常の再生に戻ります。

## 7. 作業を保存する

プロジェクトは編集のたびにブラウザのローカルストレージへ自動保存されます。**SYSTEM → SAVE AS** で名前付きスナップショットを作成でき、**EXPORT** で JSON バックアップをダウンロードできます。

> **ヒント**: プロジェクトはこのブラウザのローカルにのみ保存されます。ブラウザのデータを消去する前に、必ず Export でバックアップを取ってください。
