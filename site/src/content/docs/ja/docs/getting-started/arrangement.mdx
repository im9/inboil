パターンは**絵の具**、シーンは**キャンバス**だと思ってください。マトリックスで色を混ぜて、キャンバスに配置し、つないで再生フローを作ります。

## 1. 絵の具を混ぜる

**マトリックス**はあなたのパレットです。各セルがひとつのパターン — タップで選択、ダブルタップでステップエディターを開いてビートを打ち込みます。色がついたセルにはデータが入っていて、空のセルは透明です。

左の色つきセルをタップしてみてください — 右のステップグリッドにパターンの中身が表示されます:

<PlaygroundMatrixView client:visible />

アプリではまずここで複数のパターンを作ります: Verse のグルーヴ、Chorus のバリエーション、Break のスパースなリズム。それぞれが、あとでキャンバスに置く「色」です。

## 2. キャンバスに描く

色ができたら、キャンバスに配置しましょう。アプリでは**マトリックスのセルをドラッグ**して**シーンにドロップ** — 離した場所にノードが現れます。ノードのエッジハンドルから別のノードへドラッグすると接続が作られます。

4つのパターンが分岐のあるグラフに接続されています: Verse → Chorus → Break *または* Break2 → Verse。エンジンがどちらの Break を再生するかはランダムです。ノードをドラッグして動かしたり、左のマトリックスからパターンをシーンにドラッグしてみてください:

<PlaygroundSceneView client:visible />

## 3. 流れに任せる

上のルートノード（ドット付き）の再生ボタンを押すと、エンジンがグラフをたどって自動的にパターンを切り替えます。分岐点ではランダムに次のノードが選ばれる — だからアレンジは毎回違います。

| モード | 動作 |
|--------|------|
| **Loop** | 選択中のパターンをリピート（デフォルト） |
| **Scene** | ルートノード（ドット付き）からグラフをたどって再生 |

アプリではルートノードの再生ボタンで Scene モード再生が始まります。出力エッジのないノードに到達すると再生が止まります。
