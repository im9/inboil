## シーンとは？

シーンは**パターンをつないだグラフ**です。直線的なタイムラインではなく、パターンがノードとして接続されています。あるパターンの再生が終わると、エッジをたどって次のパターンへ進みます。

これにより、分岐したり、ループしたり、予想外の展開が生まれるアレンジが可能に — 従来の DAW よりも、ライブ DJ の感覚に近い構成ができます。

## パターンマトリックス

左側の**マトリックス**にはパターンの一覧が表示されます。各セルがひとつのパターンに対応しており、明るさが密度（アクティブなステップが多いほど明るい）、色味がパターンの色を示します。

タップで選択、ダブルタップでステップエディターが開きます。

## シーングラフ

シーンビューでは、パターンを再生グラフとして接続します。ノードをドラッグして配置を変更。ノードやエッジをクリックして選択できます。

## シーンを構成する

### ノードの追加

Scene ツールバーの **+** ボタンをタップしてパターンノードを追加します。各ノードはパターンへの参照です。**マトリックスからドラッグ**してシーンに追加することもできます。

### ノードの接続

**ノードのエッジからドラッグ**して別のノードにつなぐと、接続が作成されます。エッジは再生の進行先を定義します。

### エッジの重み付け

ノードに複数の出力エッジがある場合、**重み**に基づいてランダムに次のノードが選ばれます。重みが大きいほど選ばれやすくなります。これにより、繰り返さないジェネレーティブなアレンジが生まれます。

### ジェネレーティブノード

空白部分を右クリック（またはロングプレス）すると、**ジェネレーティブノード**を追加できます:

- **Turing Machine** — 確率的なパターン生成
- **Quantizer** — スケールに合わせたピッチクオンタイズ
- **Tonnetz** — トーンネットワーク上のハーモニーナビゲーション

ジェネレーティブノードは、シーン再生中にリアルタイムで新しい音楽素材を生成します。

## デコレーター

ノードには再生動作を変更する**デコレーター**を付けられます:

| デコレーター | 効果 |
|--------------|------|
| **Transpose** | ピッチを上下にシフト（相対）またはキーを固定（絶対） |
| **Tempo** | このパターンの BPM を変更 |
| **Repeat** | 次に進む前にパターンを N 回繰り返す |
| **FX** | このセクションのエフェクトを有効/無効化 |
| **Automation** | パターンの再生時間に沿って任意のパラメーターを自動変化 |

シーンノードを選択した状態で **Dock パネル**からデコレーターを追加できます。詳細は[デコレーター](/ja/docs/scene/decorators/)をご覧ください。

## 再生モード

| モード | 動作 |
|--------|------|
| **Loop** | 現在のパターンをループ再生（デフォルト） |
| **Scene** | ルートノードからシーングラフをたどって再生 |

- **ルートノード**（ドット付き）がシーン再生の開始点です
- エッジをたどりながら、分岐点では重みに応じてランダムに進みます
- 出力エッジのないノードに到達すると再生が停止します（ターミナルノード）

再生ボタンのタップでループモード、長押しでシーン再生に切り替えられます。
