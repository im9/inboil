## 概要

ステップグリッドの各トラック下部にモードバーがあり、**STEP**・**VEL**・**CHNC** の 3 つの表示を切り替えられます。ステップごとにダイナミクスと確率を細かくコントロールできます。

<PlaygroundStepGrid client:visible />

## 3 つのモード

### STEP モード（デフォルト）

標準の表示です。タップでステップのオン/オフを切り替え、ビートを組み立てます。

### VEL モード（Velocity）

アクティブな各ステップに**縦バー**が表示されます。上下にドラッグして、個々のヒットのベロシティ（音量）を設定します。

- ベロシティが高い = 音が大きい
- 範囲: 0〜127（パーセンテージで表示）
- ベロシティに変化をつけると、グルーヴ感や人間味が出る

**ヒント**: ダウンビート（ステップ 0、4、8、12）を高いベロシティでアクセントし、裏拍を低いベロシティでゴーストノートにすると、自然なノリになります。

### CHNC モード（Chance / 確率）

アクティブな各ステップに**パーセンテージ**が表示されます。上下にドラッグしてトリガー確率を設定します。

- **100%** — 必ず発音（デフォルト）
- **50%** — 半分の確率で発音（コイントス）
- **0%** — 発音しない
- 中間の値 — 割合に応じた確率

Chance を使うと**ジェネラティブな変化**が生まれます。同じパターンでもループごとに違って聴こえます。Scene グラフの重み付きエッジと組み合わせれば、まったく同じ繰り返しにならないアレンジを構築できます。

### 実践テクニック

| テクニック | やり方 |
|-----------|--------|
| **ゴーストノート** | ハイハットの裏拍を 30〜50% に設定 |
| **ランダムフィル** | 全ステップにフィル用ヒットを 15〜25% で追加 |
| **変化するグルーヴ** | 80% と 60% をパターン全体に散りばめる |
| **スパースなテクスチャ** | 全ステップを 10〜20% にしてアンビエントパーカッションに |

## ステップ数

トラック右端の**ステップ数**をタップすると変更できます。範囲は **2〜64 ステップ**。トラックごとに異なるステップ数を設定すると、ポリリズムが生まれます。
