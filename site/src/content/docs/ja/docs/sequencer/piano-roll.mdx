## 概要

Piano Roll は**メロディトラック**（BASS、LEAD、FM、WAVE）で自動的に開きます。左側に鍵盤、メイン領域にステップ×ピッチのグリッドが表示され、各セルが特定のステップ・ピッチのノートに対応します。

<PlaygroundPianoRoll client:visible />

## 操作方法

| 操作 | 結果 |
|------|------|
| 空セルを**タップ** | ノートを配置 |
| 既存ノートを**タップ** | ノートを削除 |
| ノートを**縦にドラッグ** | ピッチを変更 |
| ノートを**横にドラッグ** | ステップ位置を変更 |
| **ノートの端をドラッグ** | デュレーションを変更 |

## ポリフォニー

同じステップに複数のノートを置くとコードになります。同時発音数はボイスのポリモードによって決まります：

- **MONO** — 単音（最後に置いたノートが優先）
- **POLY8** — 最大 8 音同時発音
- **WIDE4** — 4 ボイス＋ステレオスプレッド
- **UNISON** — 全ボイスが同じノート（太いサウンド）

## Slide（グライド）

**SLD** を有効にすると、連続するノート間でスムーズなピッチグライドがかかります。アシッドベースラインには必須の機能で、303 スタイルの BASS ボイスは特徴的なポルタメントで反応します。

## Scale モード

**Scale モード**を有効にする（KEY/SCALE ボタン）と、選択したスケール内のノートだけに配置が制限されます。利用可能なモード：

| モード | 音程パターン |
|--------|-------------|
| **Ionian**（メジャー） | 全-全-半-全-全-全-半 |
| **Dorian** | 全-半-全-全-全-半-全 |
| **Phrygian** | 半-全-全-全-半-全-全 |
| **Lydian** | 全-全-全-半-全-全-半 |
| **Mixolydian** | 全-全-半-全-全-半-全 |
| **Aeolian**（マイナー） | 全-半-全-全-半-全-全 |
| **Locrian** | 半-全-全-半-全-全-全 |

スケール外のノートはグリッド上で薄く表示されますが、クロマチックな経過音が必要な場合はそのまま入力できます。

## Chord Brush

**Chord Brush** を使うと、1 タップでコード全体を配置できます：

- **Triad** — ルート + 3rd + 5th
- **7th** — ルート + 3rd + 5th + 7th
- **sus2** — ルート + 2nd + 5th
- **sus4** — ルート + 4th + 5th

Scale モードと組み合わせると、Chord Brush は現在のキーに合った正しいインターバルを自動的に使います。間違ったコードを置いてしまう心配がありません。
