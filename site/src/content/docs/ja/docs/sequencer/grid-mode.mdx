## 概要

Grid モードはシーケンサーのデフォルト表示です。行がトラック、列がステップに対応しています。セルをタップしてオン/オフを切り替えられます。ドラッグすれば一度に複数ステップをまとめて打ち込めます。

Grid と Tracker の入力モード切り替えは **Settings → PATTERN INPUT** から。

## トラックヘッダ

各トラック行の左側にヘッダがあり、5 つの要素が並んでいます：

| 要素 | 操作 |
|------|------|
| **トラック名** | タップでベロシティレーンを展開 |
| **ステップ数**（数字） | タップで切り替え、長押しでグリッド上から直接選択（2〜64） |
| **ステップスケール**（例：1/16） | タップでステップ解像度を切り替え（1/8 → 3/16 → 1/16 → 3/32 → 1/32） |
| **S** | トラックのソロ / ソロ解除 |
| **M** | トラックのミュート / ミュート解除 |

## 操作方法

| 操作 | 結果 |
|------|------|
| セルを**タップ** | ステップのオン/オフ切り替え |
| セルを**ドラッグ** | 複数ステップの一括入力 |
| **Ctrl+C** / **Ctrl+V** | パターンのコピー / 貼り付け |
| **Delete** | 現在のパターンの全ステップをクリア |
| **Ctrl+Z** / **Ctrl+Shift+Z** | 元に戻す / やり直し |

## ステップ数とポリリズム

各トラックは独立したステップ数を持っています。デフォルトは 16 ステップですが、**2〜64** の範囲で自由に設定できます。2〜16 の全整数に加え、24、32、48、64 が選択可能です。

**ステップ数の設定方法**: 数字をタップで順送り、または**長押し**（300ms ホールド）でステップ設定モードに入ると、グリッドがビジュアルピッカーに変わり、タップで直接ステップ数を指定できます。

トラックごとにステップ数が異なると、それぞれ別の周期でループし**ポリリズム**が生まれます。例えば：

- キック: 16 ステップ（通常の 4/4）
- ハイハット: 12 ステップ（キックに対して 3/4）
- パーカッション: 7 ステップ（奇数ループで変化し続けるパターン）

グリッドは各トラックのステップ数に合わせて伸縮表示されます。

## ステップスケール

各トラックは独立した**ステップスケール**（各ステップの時間解像度）を設定できます。デフォルトは **1/16**（標準的な 16 分音符）です。選択可能なスケール：

| スケール | 速度 | 用途 |
|----------|------|------|
| **1/8** | 半速 | ゆったりとしたパターン |
| **3/16** | 付点 16 分 | 3 連符に近いフィール |
| **1/16** | 通常（デフォルト） | 標準的な 16 分音符 |
| **3/32** | 付点 32 分 | 16 分と 32 分の中間 |
| **1/32** | 倍速 | 高速ロール、細かいハイハット |

**ステップスケールの設定方法**：トラックヘッダのスケール表示（例：`1/16`）をタップして切り替えます。

異なるステップ数と異なるステップスケールを組み合わせることで、リッチな**ポリメトリック**テクスチャーを作れます。例えば：

- キック：16 ステップ、1/16（通常の 4/4）
- ハイハット：16 ステップ、1/32（倍速の 16 分パターン）
- パーカッション：12 ステップ、1/16（3/4 ポリリズム）

各トラックのプレイヘッドはそれぞれ独立した速度で進みます。

## パターンツールバー

パターンエディタ上部のツールバーには以下のコントロールがあります：

- **TMPL** — トラックテンプレートを適用
- **KEY** — ルートノートを選択
- **OCT** — オクターブオフセットを調整
- **RND** — パターンをランダム化
- **VKBD** — バーチャルキーボードの表示切り替え
- **LOOP** — このパターンをループ

パターンのリネームや色の変更は **Dock パネル**（左側）から行えます。
