## 概要

Grid モードはシーケンサーのデフォルト表示です。行がトラック、列がステップに対応しています。セルをタップしてオン/オフを切り替えられます。ドラッグすれば一度に複数ステップをまとめて打ち込めます。

Grid モードへの切り替えは **SYSTEM → GRID** から（初期状態ではデフォルトで選択されています）。

## 操作方法

| 操作 | 結果 |
|------|------|
| セルを**タップ** | ステップのオン/オフ切り替え |
| セルを**ドラッグ** | 複数ステップの一括入力 |
| **トラック名をタップ** | トラック選択 → Dock パネルが開く |
| **ステップ数をタップ**（右端） | ステップ数を変更（2〜64） |

## ステップ数とポリリズム

各トラックは独立したステップ数を持っています。デフォルトは 16 ステップですが、**2〜64** の範囲で自由に設定できます。

トラックごとにステップ数が異なると、それぞれ別の周期でループし**ポリリズム**が生まれます。例えば：

- キック: 16 ステップ（通常の 4/4）
- ハイハット: 12 ステップ（キックに対して 3/4）
- パーカッション: 7 ステップ（奇数ループで変化し続けるパターン）

グリッドは各トラックのステップ数に合わせて伸縮表示されます。

## パターン操作

**Matrix**（左パネル）のセルをタップしてパターンを選択します。ツールバーのボタンで操作します：

- **CPY** — 現在のパターンをコピー
- **PST** — コピーしたパターンを貼り付け
- **CLR** — 現在のパターンの全ステップをクリア
- **RAND** — 全ステップをランダム化
- **Ctrl+Z** / **Ctrl+Shift+Z** — 元に戻す / やり直し

**パターン名**をクリックするとリネームできます。**カラードット**をクリックするとパターンの色を変更できます（Matrix や Scene ビューで表示されます）。
