## 概要

Tracker モードでは、パターンを縦方向のノートデータ一覧として表示します。往年の Amiga/PC トラッカーのようなスタイルで、各行がステップに対応し、列にはノート・ベロシティ・デュレーション・スライド・確率の値が並びます。

Grid と Tracker の入力モード切り替えは **Settings → PATTERN INPUT** から。

## 各列の意味

| 列 | 内容 | 範囲 |
|----|------|------|
| **NOTE** | ノート名（C-4、D#5 など）— ドラムトラックは `***` 表示 | C-3 〜 B-4 |
| **VEL** | ベロシティ（音量） | 00〜99（00 = 100%） |
| **DUR** | デュレーション（ステップ単位） | 01〜16 |
| **SLD** | スライド/グライド（次のノートへの滑り） | SLD / --- |
| **CHN** | 確率（発音する確率） | 10%〜90%（--- = 100%） |

## ノート入力

文字キー（**C D E F G A B**）でノートを入力できます。**#** を続けるとシャープになります。オクターブは同じトラックの直前のノートに自動で合わせられます。

- **Shift+上 / Shift+下** — ノートを ±1 オクターブ移調

## ナビゲーション

| キー | 操作 |
|------|------|
| **上 / 下矢印** | カーソル行を移動 |
| **左 / 右矢印** | カーソル列を移動 |
| **PageUp / PageDown** | ±8 行ジャンプ |
| **[ / ]** | 前 / 次のトラックに切り替え |
| **Enter** | ステップのオン/オフ切り替え（SLD 列ではスライド切り替え） |
| **Delete / Backspace** | ステップをクリア |
| **0〜9** | 値を直接入力（VEL・DUR・CHN 列） |

## Tracker モードが向いている場面

Tracker モードは以下のような用途に最適です：

- **正確なノート入力** — グリッドをクリックする代わりに、ノート値を直接タイプ
- **ベロシティの細かい調整** — 数値で正確にベロシティを指定
- **スライドパターン** — アシッドベースラインに欠かせないスライドをステップごとに簡単設定
- **データの一覧性** — 1 トラックの全データをコンパクトに俯瞰
