## 概要

Tracker モードでは、パターンを縦方向のノートデータ一覧として表示します。往年の Amiga/PC トラッカーのようなスタイルで、各行がステップに対応し、列にはノート・ベロシティ・デュレーション・スライド・チャンネルの値が並びます。

Tracker モードへの切り替えは **SYSTEM → TRKR** から。

## 各列の意味

| 列 | 内容 | 範囲 |
|----|------|------|
| **NOTE** | MIDI ノート名（C-4、D#5 など） | C-0 〜 G-10 |
| **VEL** | ベロシティ（音量） | 00〜99 |
| **DUR** | デュレーション（ティック単位） | 01〜99 |
| **SLD** | スライド/グライド（次のノートへの滑り） | ON/OFF |
| **CHN** | MIDI チャンネル | 01〜16 |

## ナビゲーション

| キー | 操作 |
|------|------|
| **矢印キー** | セル間のカーソル移動 |
| **Enter** | 選択セルの編集 |
| **Escape** | 編集キャンセル / 選択解除 |
| **Delete** | セルの値をクリア |
| **Tab** | 次の列へ移動 |

## Tracker モードが向いている場面

Tracker モードは以下のような用途に最適です：

- **正確なノート入力** — グリッドをクリックする代わりに、ノート値を直接タイプ
- **ベロシティの細かい調整** — 数値で正確にベロシティを指定
- **スライドパターン** — アシッドベースラインに欠かせないスライドをステップごとに簡単設定
- **データの一覧性** — 1 トラックの全データをコンパクトに俯瞰
