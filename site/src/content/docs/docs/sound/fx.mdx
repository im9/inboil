## FX Pad

The FX Pad provides **4 send effects** arranged as draggable XY pads. Each effect has 3 flavours — tap the effect name to cycle through them.

<PlaygroundFxPad client:visible />

### Effects

| Effect | Flavours |
|--------|----------|
| **Reverb** | Room / Hall / Shimmer |
| **Delay** | DIGI (ping-pong) / DOT.8 (dotted 8th) / TAPE (warm LP-filtered) |
| **Glitch** | CRUSH (bitcrush) / REDUX (downsample) / STTR (stutter) |
| **Granular** | CLOUD (grain shower) / FREEZE (auto-freeze) / STRCH (timestretch) |

### Per-Track Sends

Each track has independent send levels for each effect (VERB, DLY, GLT, GRN knobs in the Dock panel). Drag up to increase the send amount.

### XY Control

- **X axis** — time/size (reverb size, delay time, grain size)
- **Y axis** — character/feedback (reverb damping, delay feedback, grain density)
- **Tap** the pad to toggle the effect on/off
- **Long-press** (400 ms) any pad to open a **bubble menu** for switching flavours
- On the Granular pad, long-press toggles **freeze** when the effect is already on

## EQ

The EQ provides a **3-band equalizer** plus a filter sweep node.

<PlaygroundFilterView client:visible />

### Bands

| Band | Description |
|------|-------------|
| **LOW** | Bass — peaking or shelf (double-tap to toggle) |
| **MID** | Mid — peaking EQ |
| **HIGH** | Treble — peaking or shelf (double-tap to toggle) |
| **Filter** | LP/HP sweep |

Drag the nodes to adjust gain and frequency.

## Master

The Master section controls the final output stage.

<PlaygroundMasterView client:visible />

### XY Pads

| Pad | X Axis | Y Axis |
|-----|--------|--------|
| **COMP** | Threshold | Ratio |
| **DUCK** | Depth | Release |
| **RET** | Reverb return level | Delay return level |

### VU Meter

The VU meter shows real-time **L/R peak levels** and gain reduction (GR). Keep peaks below 0 dB to avoid clipping.

## Insert FX

Each track has a **2-slot serial insert chain** — processed inline before send effects. Slot 1 feeds into Slot 2 (Voice → Slot 1 → Slot 2 → Sends/Mix).

| Setting | Options |
|---------|---------|
| **Type** | OFF / Reverb / Delay / Glitch |
| **Flavour** | Same as send FX (Room/Hall, Digital/Dotted/Tape, Bitcrush/Redux) |
| **MIX** | Dry/wet blend (0–100%) |
| **X / Y** | Type-dependent (e.g. Size/Damp for reverb, Time/FB for delay) |

Each slot is independent — combine Reverb → Delay, Delay → Glitch, or any other pairing. Per-step **P-Locks** are supported for all insert parameters (MIX, X, Y per slot) via the **INS** tab in the velocity lane.

Set both slots to OFF to collapse the UI to a single row.

## Signal Flow

```
Voice → Insert FX (Slot 1 → Slot 2) → Per-Track Sends ──► Reverb / Delay / Glitch / Granular
                                                                    │
                                                              FX Return (RET)
                                                                    ↓
Voice ──► Sidechain Ducker ──► Bus Compressor ──► 3-Band EQ
              ──► DJ Filter ──► Break Gate ──► Master Gain
              ──► Peak Limiter ──► Output
```
