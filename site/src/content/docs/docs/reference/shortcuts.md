## Global

| Shortcut | Action |
|----------|--------|
| **Space** | Play / Stop |
| **Ctrl+Z** | Undo |
| **Ctrl+Shift+Z** | Redo |
| **Escape** | Close sheet / Deselect |
| **Delete** | Remove selected node or edge |
| **Ctrl+C** | Copy pattern |
| **Ctrl+V** | Paste pattern |

## Step Grid

| Shortcut | Action |
|----------|--------|
| **Click** cell | Toggle step on/off |
| **Drag** across cells | Paint multiple steps |
| **Click** track name | Select track (open in Dock) |
| **Click** step count | Cycle step count (2 / 4 / 8 / 16 / 32 / 64) |
| **Shift+Drag** velocity bar | Switch between velocity and chance editing |

## Tracker Mode

| Shortcut | Action |
|----------|--------|
| **Arrow keys** | Navigate between cells |
| **Enter** | Toggle step on/off (SLD column: toggle slide) |
| **Delete / Backspace** | Clear step |
| **PageUp / PageDown** | Jump 8 rows up/down |
| **[ / ]** | Switch to previous/next track |
| **0–9** | Set velocity (VEL column) or chance (CHN column) |
| **C–B** | Enter note name (melodic tracks, NOTE column) |
| **#** | Toggle sharp/flat |
| **Shift+Up / Shift+Down** | Octave up/down |

## Piano Roll

| Shortcut | Action |
|----------|--------|
| **Click** empty cell | Place note |
| **Click** existing note | Remove note |
| **Drag** note vertically | Change pitch |
| **Drag** note horizontally | Change step position |
| **Drag** note edge (right) | Change duration |
| **Right-click** | Erase note (in Draw mode) |
| Hold **D** | Temporarily switch to Draw mode |
| Hold **E** | Temporarily switch to Eraser mode |
| Hold **C** | Temporarily switch to Chord mode |
| **▲ / ▼** or **Scroll** | Shift octave window |

## Scene View

| Shortcut | Action |
|----------|--------|
| **Click** node | Select node |
| **Drag** node | Reposition |
| **Click** edge | Select edge |
| **Drag** node handle (right side) | Start edge-draw (drag to target) |

| **Right-click** empty space | Bubble menu (add node/label) |
| **Delete** | Remove selected element |
| **Escape** | Deselect all |
| **Ctrl+C** | Copy node/subgraph |
| **Ctrl+V** | Paste |
| **Arrow Up / Down** (edge selected) | Reorder edge (change playback order) |
| **Scroll / Pinch** | Zoom |
| **Middle-drag** or **Ctrl+Drag** | Pan canvas |
| **Alt+A / D / W / S** (2+ nodes) | Align left / right / top / bottom |
| **Alt+X / E** (2+ nodes) | Center horizontally / vertically |
| **Alt+Shift+X / E** (2+ nodes) | Distribute horizontally / vertically |

## Virtual Keyboard (VKBD)

Enable with the **VKBD** button in the pattern toolbar.

| Key | Note |
|-----|------|
| **A S D F G H J K L ;** | C D E F G A B C D E (chromatic) |
| **Z** | Octave down |
| **X** | Octave up |
| **1–0** | Velocity (1=low, 0=max) |

## MIDI

USB/BLE MIDI keyboards are supported via the Web MIDI API. Configure input device and channel routing in **SYSTEM → MIDI**.
