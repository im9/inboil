## Global

| Shortcut | Action |
|----------|--------|
| **Space** | Play / Stop |
| **Ctrl+Z** | Undo |
| **Ctrl+Shift+Z** | Redo |
| **Escape** | Close panel / Deselect |
| **Delete** | Remove selected node or edge |
| **Ctrl+C** | Copy pattern |
| **Ctrl+V** | Paste pattern |

## Step Grid

| Shortcut | Action |
|----------|--------|
| **Click** cell | Toggle step on/off |
| **Drag** across cells | Paint multiple steps |
| **Click** track label | Select track (open in Dock) |
| **Click** step count | Change step count (2–64) |

## Tracker Mode

| Shortcut | Action |
|----------|--------|
| **Arrow keys** | Navigate between cells |
| **Enter** | Edit selected cell |
| **Tab** | Move to next column |
| **Delete** | Clear cell value |
| **Escape** | Cancel edit |

## Piano Roll

| Shortcut | Action |
|----------|--------|
| **Click** empty cell | Place note |
| **Click** existing note | Remove note |
| **Drag** note vertically | Change pitch |
| **Drag** note edge | Change duration |

## Scene View

| Shortcut | Action |
|----------|--------|
| **Click** node | Select node |
| **Drag** node | Reposition |
| **Click** edge | Select edge |
| **Drag** from node edge | Create connection |
| **Right-click** empty space | Context menu (add node/label) |
| **Delete** | Remove selected element |
| **Scroll / Pinch** | Zoom |
| **Middle-drag** | Pan canvas |

## Virtual Keyboard (VKBD)

Enable with the **VKBD** button.

| Key | Note |
|-----|------|
| **A S D F G H J K L ;** | C D E F G A B C D E (chromatic) |
| **Z** | Octave down |
| **X** | Octave up |
| **1–0** | Velocity (1=low, 0=max) |

## MIDI

USB and BLE MIDI keyboards are supported. Configure input device and channel routing in **SYSTEM → MIDI**.
