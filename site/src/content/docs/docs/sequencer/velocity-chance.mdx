## Overview

Tap a **track name** in the step grid to expand the velocity lane below it. The lane shows a vertical bar for each step, and a **VEL / CHNC** toggle button lets you switch between velocity and chance editing.

<PlaygroundStepGrid client:visible />

## Velocity

Drag a bar **up/down** to set the velocity (volume) of each hit. The bar height represents the value as a percentage.

- Higher bar = louder hit
- Range: 5%–100%
- Use velocity variation to add groove and human feel

**Tip**: Accent the downbeats (steps 0, 4, 8, 12) at high velocity and ghost the off-beats at low velocity for a more natural feel.

## Chance (Probability)

Click the **CHNC** button (or **Shift+drag** a bar) to switch to chance mode. Bars turn blue. Drag up/down to set the trigger probability.

- **100%** — always triggers (default, shown when no chance is set)
- **50%** — triggers half the time
- Lower values — proportional probability

Steps with chance set below 100% show a small **blue diamond** in the bottom-left corner of the step cell.

Chance creates **generative variation** — the same pattern sounds different each loop. Combined with the Scene graph's weighted edges, you can build arrangements that never repeat exactly.

## Practical Uses

| Technique | How |
|-----------|-----|
| **Ghost notes** | Set hi-hat off-beats to 30–50% chance |
| **Random fills** | Add fill hits at 15–25% chance on every step |
| **Evolving grooves** | Mix 80% and 60% across a pattern |
| **Sparse textures** | Set all steps to 10–20% for ambient percussion |
