## 1. Paint Steps & Play

Tap cells in the step grid to toggle triggers on/off. **Drag to paint** multiple steps in one gesture. Each row is a track — kick, snare, hi-hat, etc.

This grid has a 4-on-the-floor kick and hi-hat pattern pre-loaded. Hit **▶ Play** to hear it, then try adding snare hits or toggling steps:

<PlaygroundAudioDemo client:load />

### Step Count

Each track can have **2–64 steps** independently. Tap the step count number to cycle, or **long-press** to pick visually on the grid. Odd step counts create polyrhythms — try 5 or 7 against a 16-step kick.

## 2. Tweak Sounds

Select a track in the grid to see its voice parameters as knobs below. **Drag up** to increase a value, **drag down** to decrease. Hold **Shift** for fine control, double-click to reset.

Each voice type (KICK, SNARE, BASS, LEAD, FM, WAVE) has its own parameters — see [Change Sounds](/docs/getting-started/adding-sounds/) to explore all voices and audition them.

## 3. Randomize

Hit **RAND** to generate a random pattern. Use **Ctrl+Z** to undo if you don't like it. Undo is unlimited — every edit is recorded.

## 4. Performance Controls

On **desktop**, hold the performance buttons (FILL / REV / BRK in the transport bar) for real-time effects during playback:

<PlaygroundAudioDemo client:load showPerf={true} showGrid={false} />

- **FILL** — auto drum fill while held
- **REV** — reverse playback
- **BRK** — rhythmic break / stutter

Release to restore normal playback instantly.

On **mobile**, tap the **PERF** tab in the header to open the **Perf Sheet** — a Kaoss Pad-style XY controller with four tabs: PERF, GLITCH, FILTER, and MOTION. Drag the pad to control two parameters at once (e.g. fill density + velocity). The sheet includes sensor-driven effects like accelerometer tilt and shake detection.

## 5. Save Your Work

Projects auto-save on every edit to your browser's local storage. Use **SYSTEM → SAVE AS** to create named snapshots, and **EXPORT** to download a JSON backup.

> **Tip**: Projects are stored locally in this browser only. Use Export to back up your work before clearing browser data.
