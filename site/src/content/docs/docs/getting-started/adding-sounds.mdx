## Voice Parameters

Every track has a **voice** — the instrument that plays when a step triggers. Select a track to see its parameters, tweak the knobs, then tap the keys to hear the result:

<PlaygroundVoiceDemo client:load />

### Drum Voices

| Voice | Character |
|-------|-----------|
| **KICK** | Analog-style bass drum — tune, decay, drive |
| **SNARE** | Noise + tone mix — tune, snap, decay |
| **CLAP** | Layered clap — spread, decay, tone |
| **C.HH / O.HH** | Closed/Open hi-hat — tune, decay, tone |
| **CYM** | Cymbal — tune, decay, tone |

### Synth Voices

| Voice | Character |
|-------|-----------|
| **BASS** | 303-style acid bass — cutoff, resonance, envelope, accent |
| **LEAD** | Moog-style lead — filter, envelope, detune |
| **FM** | FM synthesis — ratio, modulation index, feedback |
| **WAVE** | Wavetable — morph through waveforms |

## Envelope

Synth voices use an **ADSR envelope** to shape how the sound evolves over time. Drag the knobs to see the envelope shape change:

<PlaygroundEnvGraph client:visible />

- **ATK** (Attack) — how quickly the sound reaches full volume
- **DCY** (Decay) — how quickly it drops to sustain level
- **SUS** (Sustain) — the held level while a note is active
- **REL** (Release) — how long the tail rings after release

## Presets

Each voice comes with **factory presets**. Tap the preset name in the Dock to browse. You can also save your own — dial in a sound you like, then **long-press** the preset name to save.

### Per-Step Lock

Enable **P-LOCK** in the Dock, then select a step and turn knobs. Sound changes apply to **that step only** — the rest of the pattern keeps its original sound. This is how you create filter sweeps, accent patterns, and timbral variation within a single pattern.

Try it: toggle **P-LOCK** on, tap an active step, then tweak a knob. Hit **Play** to hear the difference. Locked parameters show a blue dot on the knob.

<PlaygroundAudioDemo client:load showLock={true} />

Tap **CLR** to clear locked parameters from a step.

## Tracks

You can have up to **16 tracks**. Tap **+** to add a new track, or use the **REMOVE** button to delete one.

- **M** — Mute (silence a track)
- **S** — Solo (hear only soloed tracks; additive)

Switch voice types with the **Dock tabs**: DRUM / BASS / LEAD / SMPL.

## Sends

Each track has **4 send effects**: Reverb, Delay, Glitch, and Granular. Turn up the send level to route a track into an effect — each track's sends are independent.

| Effect | What it does |
|--------|-------------|
| **Reverb** | Room / Hall / Shimmer |
| **Delay** | Stereo / Ping-pong / Tape |
| **Glitch** | Stutter / Bitcrush / Repeat |
| **Granular** | Freeze / Scatter / Cloud |

Each effect has 3 flavours and an XY pad for real-time control — see [Effects](/docs/sound/fx/) for details.
